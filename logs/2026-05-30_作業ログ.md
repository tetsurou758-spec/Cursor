## 実装 API エンドポイント一覧

| メソッド | パス | 認証 | 内容 |
|---|---|---|---|
| POST | `/api/login` | 不要 | ログイン認証。代理店コード+ID+パスワードを検証しJWTトークンを返す |
| GET | `/api/dashboard` | JWT必須 | 今月・翌月の更改件数をステータス別に集計して返す（代理店コードで自動フィルタ） |
| GET | `/api/maturity` | JWT必須 | 満期3カ月前〜翌3カ月の契約一覧を返す。maturity_noticesをLEFT JOIN。6種の検索パラメータ対応 |

### GET /api/maturity 検索パラメータ

| パラメータ | 型 | 説明 |
|---|---|---|
| `date_from` | string | 満期年月日FROM（YYYY-MM-DD） |
| `date_to` | string | 満期年月日TO（YYYY-MM-DD） |
| `staff_code` | string | 担当者コード |
| `policy_type` | string | 種目（自動車/火災/傷害/その他） |
| `renewal_status` | string | 更改STS（未対応/対応中/更改済/落ち） |
| `followcall_status` | string | フォローコールSTS（未実施/実施済） |
